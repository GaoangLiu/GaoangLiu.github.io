Python Code to get the day.

class MyDate:
    def dayOfTheWeek(self, d, m, y):
        if m < 3:
            m += 12 
            y -= 1
        c, y = divmod(y, 100)
        res = (d + (13 * (m + 1)) // 5  + y + y // 4 + c // 4 - 2 * c) % 7 
        days = ["Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"]
        return days[(res + 6) % 7]
